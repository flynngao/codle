## Codle

### Read Code in Kindle

    npm install codle -g

### config Your Email & kindle

    codle config -u your-email -p passw -k kindle
    
### config Your Kindle

    codle -k kindle-email 

### codle push 

    codle push -g git-address

    codle push -d documents

    codle push -f sigle file path
